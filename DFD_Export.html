Data Store Definitions

Data Store No. Data Store Name Description
D1UsersStores admin, teacher, and student account details, approval status, role, OTP and profile info
D2CoursesStores course records created by teachers
D3ChaptersStores chapters belonging to courses
D4EnrollmentsStores student-course enrollment records
D5QuizzesStores quiz records linked to chapters
D6Questions & AnswersStores quiz questions and answer options
D7Quiz AttemptsStores student quiz attempts, selected answers, and scores
D8ArticlesStores writing competition submissions, approval, and winner status
D9Live Stream SessionsStores live class session details and YouTube stream data
D10Session DataStores temporary session information such as pending verification user id
